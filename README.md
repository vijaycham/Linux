## Arrow keys
right and left - move the cursor
up and down - will move to the last command

##Command Structure
> command - options arguments

ex:echo will echo next word
- ncal july 1969
- ncal 1969 
- sort colors.txt
- rm colors.txt - it deletes

## options

> command -option
> ncal -h highlight off
> ncal -j Julian calender
> ncal -M  starts with monday

## combining options

>ncal -3 -h
>ncal -3Mj

## Long Form options

> date -u
> date --universal
> sort color.txt -r
> sort color.txt --reverse
> sort color.txt -r -u (for unique)
> sort solor.txt --reverse --unique

## Options With Parameters
>ncal -A2 after two months
>ncal -B2 before two months

- All together
>ncal july 1969 -B1 -M
